---

### **Features Overview**

|**Feature**|**Brief Description**|
|---|---|
|**Full Page Cache**|Server-level cache for ultra-fast page load times.|
|**Private & ESI Caching**|Edge Side Includes allow partial caching for dynamic content.|
|**Image Optimization**|Compresses and converts images to WebP & AVIF formats.|
|**Critical CSS & CSS/JS Optimization**|Removes unused CSS, combines/minifies assets, async loading.|
|**Object Cache**|Supports Redis / Memcached / LiteSpeed MemCache engines.|
|**QUIC.cloud CDN Integration**|Global acceleration with smart edge delivery & DDoS protection.|
|**WooCommerce Optimization**|Cart fragments, checkout, and dynamic page optimization.|
|**Database Cleaner**|Removes post revisions, transients, and unused tables.|
|**Crawler / Auto-Warm Cache**|Rebuilds cache automatically to avoid cold-load delays.|
|**HTACCESS & Server-level Controls**|Deep configuration directly at server layer.|
