# **How to Approach : Event Tickets (Ticketing & RSVP System for Events)**

> **Senior Developer Guidance:**  
> **Event Tickets** is a plugin built to integrate with **The Events Calendar** for selling tickets, managing RSVPs, and handling event attendee records directly inside WordPress.  
> It enables creators to **sell paid tickets via WooCommerce or accept free registrations**, generate attendee lists, and automate confirmation workflows â€” without needing third-party ticketing platforms.

Use Event Tickets when the project requires **event ticketing, seat reservation, attendee tracking, or RSVP management** connected to a calendar system.

---

### **Features Overview**

|**Feature**|**Brief Description**|
|---|---|
|**RSVP & Free Registration System**|Allow users to register attendance without payment.|
|**Paid Ticket Sales via WooCommerce**|Sell tickets securely with Stripe, PayPal, RazorPay, etc.|
|**Attendee Management Dashboard**|Track participants, check-in status & booking details.|
|**Custom Ticket Types**|Create multiple ticket categories (VIP, Early Bird, Student etc.).|
|**Email Notification & Ticket Delivery**|Automated confirmation & digital ticket sendouts.|
|**QR Code Check-In (Pro)**|Scan QR codes for fast attendance verification.|
|**Tickets Embedded in Event Pages**|Inline ticket purchase UI integrated with event descriptions.|
|**Stock & Availability Limits**|Control number of seats/tickets per category.|
|**Export & Reporting Tools**|Export attendee lists for onsite event management.|
|**Calendar & Schedule Integration**|Works seamlessly with The Events Calendar layout system.|
